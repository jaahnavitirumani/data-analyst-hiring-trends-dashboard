# Data Analyst Hiring Trends Dashboard

## Project Overview

The objective of this project is to analyze Data Analyst job postings and uncover hiring trends, in-demand skills, geographic opportunities, industry demand, and employer characteristics. Using Python for data analysis and Power BI for visualization, this project provides insights into the current Data Analyst job market.

## Business Problem

Students and aspiring Data Analysts often struggle to identify which skills, industries, and locations offer the best career opportunities. This project aims to answer key questions such as:

* Which companies hire the most Data Analysts?
* Which locations have the highest demand?
* What skills are most frequently requested by employers?
* Which industries actively recruit Data Analysts?
* How do company characteristics influence hiring trends?

## Key Findings

* SQL emerged as one of the most frequently requested skills.
* Python appeared consistently across Data Analyst job descriptions.
* Hiring demand was concentrated in major metropolitan locations.
* Technology and consulting industries demonstrated strong hiring activity.
* Larger organizations accounted for a significant share of job postings.
